Этот проект - пособие для разработчика, чтобы разобраться и вспомнить различные инфраструктурные технологии, способы их применения, теорию и практику для подготовки к собеседованиям. Модуль должен содержать упражнения и теорию для следующих концепций - Observability, Logs, Metrics, Containerization and Orchestration. Используй Docker, Kubernetes, Helm, Prometheus + Grafana, любой инструмент для логирования

Теория должна быть максимально проработанная, структурированная по отдельным md файлам. Каждый модуль должен содержать основные вопросы, которые спрашивают на собеседованиях для РАЗРАБОТЧИКОВ, а не DevOps - то есть не слишком поверхностно, но и не так глубоко, как для SRE.
