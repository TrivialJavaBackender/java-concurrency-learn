# Concurrency Interview Prep

## Проект
Этот проект — площадка для практики Java Concurrency (Kotlin + Java concurrency APIs).

## Команды

### Прогресс
Когда пользователь говорит "прогресс", "статус" или "как дела" — читай PROGRESS.md и покажи текущий статус.

### Проверка упражнения
Когда пользователь говорит "проверь Ex01" (или любой номер):
1. Прочитай соответствующий файл из src/main/kotlin/exercises/
2. Проверь что TODO заменены на реализацию
3. Попробуй скомпилировать: `mvn compile -q`
4. Если компилируется — запусти: `mvn exec:java -Dexec.mainClass="exercises.ExXX_NameKt" -q`
5. Проверь корректность вывода
6. Обнови PROGRESS.md

### Следующий модуль
Когда пользователь говорит "следующий" или "next" — найди первый незавершённый модуль в PROGRESS.md, покажи теорию из ROADMAP.md и предложи начать упражнения.

### Квиз
Когда пользователь говорит "квиз" или "quiz" — задай 5 случайных вопросов из INTERVIEW_QUESTIONS.md, жди ответа, оцени.

## Сборка
- `mvn compile` — компиляция
- `mvn exec:java -Dexec.mainClass="exercises.Ex01_ThreadBasicsKt"` — запуск упражнения
